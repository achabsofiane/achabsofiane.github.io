# Sofiane Achab — Portfolio

Personal portfolio of **Sofiane Achab**, Senior Product Manager & UX Designer.

🔗 **Live site:** [achabsofiane.github.io](https://achabsofiane.github.io)

## Overview

A fast, lightweight single-page portfolio showcasing product & design work
(L'Oréal, Osme, Partoo, Leadscore, and more). Built from scratch with no
framework or build step — just a single `index.html`.

## Features

- 🎨 **Pixel-faithful design** based on Figma mockups
- 📱 **Fully responsive** — desktop, tablet and mobile layouts
- 🌐 **Bilingual EN / FR** with an in-page language switch (choice remembered via `localStorage`)
- ✨ **Subtle entrance animations** on scroll
- ⚡ **Zero dependencies** — plain HTML, CSS and vanilla JavaScript

## Run locally

No build needed. Either:

- Open `index.html` directly in your browser, **or**
- Serve the folder (recommended, so images load reliably):

```bash
python3 -m http.server 3000
# then open http://localhost:3000
```
